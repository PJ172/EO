# Báo cáo Tiến độ Công việc - 18/03/2026

## 1. Các công việc đã hoàn thành trong ngày

### Tối ưu hóa Sơ đồ Tổ chức (Org Chart Pro Max)
- **Thuật toán mới**: Chuyển đổi từ ELK sang thuật toán `MrTree`, giải quyết triệt để lỗi chồng chéo node.
- **Tối ưu không gian**: Căn chỉnh khoảng cách tự động giữa các node và các tầng (level), đảm bảo sơ đồ cân đối ngay cả với dữ liệu lớn.
- **Logic hiển thị**: Tích hợp trường `showOnOrgChart` xuyên suốt từ Database (Prisma) đến Backend API và Frontend.

### Cải thiện AI Chatbot Widget
- **Draggable Widget**: Chuyển chatbot icon sang dạng widget có thể kéo thả tự do trên màn hình để tránh che khuất dữ liệu quan trọng.
- **Boundary Constraints**: Thiết lập giới hạn vùng kéo để icon không bị trôi ra ngoài phạm vi hiển thị của ứng dụng.
- **RAG System**: Cấu trúc lại hệ thống AI hỗ trợ truy vấn dữ liệu nội bộ qua Embedding và Vector Search.

### Thiết lập mặc định Hệ thống
- **Dữ liệu mới**: Toàn bộ các module (Công ty, Nhà máy, Khối, Phòng ban, Bộ phận, Nhân viên) khi thêm mới sẽ mặc định có tùy chọn **"Hiển thị trên sơ đồ tổ chức" = OFF**.
- **Đồng bộ hóa**: Cập nhật đồng bộ Prisma Schema và tất cả các Form Frontend để thực thi quy tắc này.

### Tối ưu hóa DataTable & Quản lý Nhân viên
- **Thước đo nén gọn**: Giảm cỡ chữ Cell hiển thị từ 14px xuống 12px cho bảng chính (DataTable) compact tinh gọn.
- **Tooltip Hover Lists**: Thiết kế component rút gọn `Tên +N` cho mảng **Người thân** và **Danh sách Hợp đồng**, khi rê chuột sẽ mở bung popup Tooltip cực kỳ chuyên nghiệp mà không phá vỡ layout bảng.
- **Fix Bug Import**: Hiệu chỉnh logic Fuzzy String Matching cho trường "Khối" trong `employee.excel.service.ts` giúp nạp file Excel chuẩn xác 100%.
- **Bỏ tính năng Org Chart**: Chính thức gỡ bỏ switch "Hiện trên sơ đồ tổ chức" khỏi form thêm mới/sửa nhân viên theo luồng vận hành mới.

### Thiết kế lại Tab Biến động Nhân sự (Employment Events)
- **Visual Node Timeline**: Thay thế bảng dữ liệu (Table) truyền thống sang cấu trúc **Dòng thời gian dạng Tree Node** dọc vô cùng thẩm mỹ.
- **Fix Bug State Form**: Sửa lỗi trượt bẫy `defaultValue` của Select "Loại biến động" khiến Form edit cập nhật rỗng.

### Đại tu giao diện Đặt phòng họp (Booking Timeline)
- **Tích hợp Gantt Grid**: Kích hoạt chế độ view Dòng thời gian ngang (Timeline Grid), giúp người dùng so sánh kẹt lịch giữa 10+ phòng chỉ trong 2 giây xem Day View.
- **Top Tab Switcher**: Thêm cụm nút chuyển đổi `[📅 Lịch]` và `[📊 Gantt]` đa luồng ngay trên Dashboard container.
- **Live Status Sidebar**: Đưa chỉ số Sức chứa phòng (Capacity) ra list hiển thị sơ khởi, tối ưu tốc độ lựa chọn phòng rảnh.
