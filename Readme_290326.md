### 2. Fix lỗi Matrix Connection (Kết nối đa tuyến)
- **Nguyên nhân:** Handle ID trên các node có dạng compound (`bottom-target`, `left-source`, `right-target`, `top-source`). Khi user kéo kết nối, các ID này bị lưu nguyên vào DB (vd: `"bottom-target:left"`). Khi load lại, ReactFlow không tìm được source handle có tên `"bottom-target"` → lỗi `Couldn't create edge for source handle id`.
- **Sửa:**
  - Thêm **handle ID normalization**: `"bottom-target"` → `"bottom"`, `"left-source"` → `"left"` trước khi lưu vào DB.
  - Sửa logic `onConnect` để mapping handle đúng giữa manager/employee.
  - Cập nhật `isMatrixHint` detection để nhận diện cả handle compound.
  - Clean up data cũ bị lỗi trong DB.
- **File:** `apps/web/src/components/org-chart/org-chart-canvas.tsx`

- **Chi tiết kỹ thuật:**

### 8. Version History — Backend
- **Mô tả:** Hệ thống lưu trữ phiên bản sơ đồ tổ chức (snapshot-based).
- **Chi tiết:**
  - **Model `OrgChartVersion`** (Prisma):
    - `chartKey` + `versionNum` composite unique
    - `isScenario` — phân biệt phiên bản vs kịch bản
    - `snapshot` (JSONB) — chứa config, overrides, viewOverrides, capturedAt

    - `createVersion()` — auto-numbering + auto-prune (50 giới hạn)

    - `deleteVersion()` — xóa đơn lẻ

    - `applyScenario()` — áp dụng kịch bản lên production
    - `buildSnapshotFromCurrentState()` — chụp trạng thái hiện tại
  - **13 API endpoints** (ORGCHART_VIEW / ORGCHART_MANAGE).

### 11. Fix Export PNG/PDF (Rewrite hoàn toàn)
- **Nguyên nhân lỗi cũ:**
  1. **Font vỡ**: `skipFonts: true` xóa font embedding → tiếng Việt bị mã hóa sai.
  2. **Bounding box sai**: `getBoundingClientRect()` trả screen coords, khi zoom ≠ 100% → nodes dồn góc phải.
  3. **UI controls lộ**: Filter thiếu `.react-flow__handle`, nút delete, badges.
  4. **PDF title sai**: `jsPDF` không hỗ trợ Unicode Vietnamese.
  5. **CORS ảnh**: Avatar images fetched cross-origin → canvas bị taint.
- **Fix:**
  - **Viewport transform parsing**: Parse CSS `matrix()` hoặc `translate()/scale()` → chuyển screen coords thành flow coordinates.
  - **Tạm set zoom=1**: `viewport.style.transform = 'translate(...)  scale(1)'` → capture → restore.
  - **Inline images**: Fetch tất cả `<img>` → convert base64 data URL trước capture, restore sau.
  - **Font handling**: `fontEmbedCSS: ''` — browser đã load font, không cần re-fetch.
  - **Filter toàn diện**: Exclude handles, design-mode buttons (.bg-rose-*, .bg-slate-*), edge text badges.
  - **PDF Vietnamese**: Render title/footer bằng **Canvas API** → export as image → `addImage()` vào PDF (bypass jsPDF Latin-1).
  - **Retry fallback**: Nếu lần 1 fail (CORS), retry với `pixelRatio: 1` + `skipFonts: true`.
- **File sửa (rewrite):**
  - `apps/web/src/components/org-chart/org-chart-export.tsx`
