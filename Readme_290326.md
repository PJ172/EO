# 📋 Changelog — 29/03/2026

## 🔧 Org Chart — Sửa lỗi & Cải tiến

### 1. Fix lỗi Database: `nodeLevels` không lưu được (HTTP 500)
- **Nguyên nhân:** Frontend gửi field `nodeLevels` trong payload nhưng bảng `org_chart_configs` chưa có cột `node_levels` trong database, và Prisma Client chưa được regenerate.
- **Sửa:** 
  - Thêm cột `node_levels JSONB` vào bảng `org_chart_configs` bằng `prisma db execute`.
  - Chạy `prisma generate` để cập nhật Prisma Client.
- **Kết quả:** Nút "Lưu toàn bộ bố cục" hoạt động bình thường, lưu được `nodeLevels`, `nodeDims`, `nodeColors` và các config khác.

---

### 2. Fix lỗi Matrix Connection (Kết nối đa tuyến)
- **Nguyên nhân:** Handle ID trên các node có dạng compound (`bottom-target`, `left-source`, `right-target`, `top-source`). Khi user kéo kết nối, các ID này bị lưu nguyên vào DB (vd: `"bottom-target:left"`). Khi load lại, ReactFlow không tìm được source handle có tên `"bottom-target"` → lỗi `Couldn't create edge for source handle id`.
- **Sửa:**
  - Thêm **handle ID normalization**: `"bottom-target"` → `"bottom"`, `"left-source"` → `"left"` trước khi lưu vào DB.
  - Sửa logic `onConnect` để mapping handle đúng giữa manager/employee.
  - Cập nhật `isMatrixHint` detection để nhận diện cả handle compound.
  - Clean up data cũ bị lỗi trong DB.
- **File:** `apps/web/src/components/org-chart/org-chart-canvas.tsx`

---

### 3. Fix lỗi 404 khi ẩn/hiện Node

- **Sửa:** Đổi thành endpoint đúng `/employees/org-chart/overrides/${chartKey}`.
- **File:** `apps/web/src/components/org-chart/org-chart-canvas.tsx` (line ~505)

### 4. Tính năng: Layer Alignment (Căn hàng theo Lớp)
- **Mô tả:** Khi click "Lưu toàn bộ bố cục", các node cùng Layer (L1, L2, ..., L10) sẽ tự động được xếp trên cùng 1 hàng ngang (cùng tọa độ Y).
- **Chi tiết kỹ thuật:**
  - Thêm post-layout pass trong `treeLayout()`:
    1. Group các node theo layer từ state `nodeLevels`.
    2. Snap tất cả node cùng layer về cùng Y (lấy Y nhỏ nhất trong nhóm).
    3. Giải quyết overlap trục X: sort theo X, đẩy node sang phải nếu chồng lấn, rồi re-center nhóm.
    4. Ngăn chặn overlap trục Y giữa các layer khác nhau (cascade gap enforcement).
  - Truyền `nodeLevels` vào hàm `treeLayout()` để biết node nào thuộc layer nào.
- **File:** `apps/web/src/components/org-chart/org-chart-canvas.tsx` (lines 131-225)
