## 4. Tối ưu Performance & Bảo mật Backend

### 4.1 Xóa bypass bảo mật hardcoded
- Loại bỏ kiểm tra hardcoded username (`'admin'`, `'it'`) trong `EmployeeQueryService.isAdministratorUser`.
- Hệ thống giờ chỉ sử dụng **Role-Based Access Control (RBAC)** nghiêm ngặt.

### 4.2 Tối ưu truy vấn Database
- Refactor `getOrgChartStructure`: chuyển từ `findMany` trong bộ nhớ sang `groupBy` cấp database → tăng tốc ~50x cho dataset lớn.
- Thêm giới hạn `take` cho các query thống kê (Dashboard, Ticket Stats, Reporting) để phòng tránh Out-of-Memory.

### 4.3 Kiểm thử tự động (Unit Testing)
- Tạo `prisma-mock.ts` factory dùng chung cho môi trường test.
- Viết **41 unit tests** cho 5 service core: `AuthService`, `EmployeeQueryService`, `TicketService`, `DashboardService`, `BookingService`.
- Kết quả: **82/104 tests pass** (3 legacy suites lỗi do dependency cũ, không phải regression).

### 4.4 CI/CD Pipeline
- Thiết lập GitHub Actions workflow: tự động chạy `type-check` và `test` trên mỗi push/PR vào `main` và `develop`.

## 6. Redesign form "Thêm thiết bị" (IT Asset Form)

### 6.1 Chuyển từ Dialog sang Sheet (Side Panel)
- **Trước**: Dialog popup modal ở giữa màn hình, nhỏ (`max-w-2xl`).
- **Sau**: Sheet panel trượt bên phải, chiếm **55% viewport** (`w-[55vw]`, max `720px`).
- Layout: Header (shrink-0) + Scrollable body (flex-1) + Sticky footer.

### 6.2 Tổ chức form theo Tab/Section
- **4 tabs chuyên biệt**: Cơ bản, Mạng & Kết nối, Phần cứng, Mua sắm & Bảo hành.
- Mỗi tab có icon, animation slide-in, và section header với icon + màu phân biệt.
- Footer có progress dots, nút "← Quay lại" / "Tiếp theo →".

### 6.3 Thông tin mới bổ sung
- **MAC Address 2** (Wi-Fi/Wireless) — lưu trong `specifications` JSON (zero-migration).
- **6 trường phần cứng**: CPU, RAM, Ổ cứng/SSD, Hệ điều hành, Kích thước màn hình, GPU.
- MAC Address input: auto-uppercase + filter chỉ ký tự hex + phân biệt Ethernet vs Wi-Fi.
- Indicator bảo hành: tự động hiển thị ✅ còn hạn / ⚠️ hết hạn.

### 6.4 Fix Combobox & UX
- Fix lỗi Radix Select `value=""` → `value={undefined}` (fix empty string bug).
- `SelectContent` thêm `position="popper"`, `sideOffset={4}`, `z-[9999]` cho dropdown đúng z-index trong Sheet.
- Tất cả placeholder text: `placeholder:text-slate-300` — rất nhạt, không nhầm với dữ liệu thực.
- Header gradient: `from-slate-900 via-slate-800 to-slate-900` + radial glow effects.
- Mẹo hữu ích cho người dùng: cách tra MAC Address, shortcut `Win + Pause` / `msinfo32`.

### 6.5 Lưu trữ dữ liệu
- Thông tin phần cứng (CPU, RAM, SSD, OS, GPU, displaySize, macAddress2) được đóng gói vào field `specifications` (JSON) của model `ITAsset`.
- **Không cần migration database** — tận dụng schema hiện có.
